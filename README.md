# 💰 Finance Dashboard

## 📌 Sobre o projeto

Dashboard moderno de controle financeiro com foco em UI/UX, responsividade e experiência do usuário.

## 🚀 Tecnologias

* HTML
* CSS
* JavaScript
* Chart.js

## 🎯 Funcionalidades

* Visualização de saldo
* Filtro de transações
* Gráfico interativo
* Dark mode
* Design responsivo
